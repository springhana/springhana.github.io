## 풀이

```python
def solution(n, control):
    answer = n
    
    for i in control:
        if i == 'w':
            answer += 1
        elif i == 's':
            answer -= 1
        elif i == 'd':
            answer += 10
        elif i == 'a':
            answer -= 10
    
    return answer
```

이 문제는 'w', 's', 'd', 'a' 에 따라서 조건 처리를 해주면 된다. 

파라미터 `n`을 결과를 반환해줄 `answer`에 담아주고 `control`을 `for`문을 이용해서 문자 하나 하나 반복해주면 된다.

`if`문이나 `switch`문을 사용해서 풀어주면 된다.
